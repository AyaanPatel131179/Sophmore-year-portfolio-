Arduino Robotics Hand

A robotic hand made from cardboard with string tendons, controlled by an Arduino. Each finger is pulled by a servo motor through a string, copying how human tendons work.

Tools I Used

  • C++ (Arduino IDE)
  • ELEGOO UNO R3 microcontroller
  • Servo motors (x2)
  • 9V Duracell battery
  • Cardboard frame & string tendons

About the Project

I wanted to build something that connects code to real movement. By copying the tendon structure of a human hand in cardboard, I could see how software running on a microcontroller turns into physical motion.

Challenges

The hardest part was calibrating the servo values so the fingers moved smoothly without ripping the cardboard joints. Getting the string tension right took a lot of testing, and I had to adjust the code and the build at the same time.

Shield Bot: Obstacle Course

A wheeled robot programmed to drive through an obstacle course on its own. It uses servo motors to move and an ultrasonic sensor to stop when it gets close to an object.

Tools I Used

  • Arduino IDE
  • ELEGOO Shield Bot
  • Servo motors
  • HC-SR04 ultrasonic sensor
  • Servo.h library

About the Project

The robot had to finish a course by itself with no remote control. It follows a set path of turns and forward moves, then stops automatically when the ultrasonic sensor sees something within 6 inches.

Challenges

Getting accurate 90-degree turns was tricky because the wheels slipped on the floor. I had to keep testing and adjusting the turn timing. The distance sensor math also needed tuning so the readings matched real life. In the future, I'd like to improve the sensor accuracy and make the turns more precise through testing and servo calibration.

Python Turtle Image Generator

A Python program that turns a photo into pixel art, drawn dot by dot using the Turtle library. It reads the brightness of each pixel and picks one of eight shades of gray.

Tools I Used

  • Python
  • Turtle graphics library
  • Pillow for image processing
  • Grayscale brightness mapping

About the Project

This project connects image processing and drawing. The program uses Pillow to read how bright each pixel is, then uses Turtle to place a colored dot for it, rebuilding the image out of gray dots.

Challenges

The main challenge while making the project was the amount of time it took to draw the image, as turtle draws one dot at a time. I turned off the screen tracer feature and only updated once at the end making the code faster. Picking the right brightness levels for each gray shade also took some trial and error as I had to manually match everything.

Shield Bot Code 
#include <Servo.h>

Servo leftServo;
Servo rightServo;

int echoPin = 10;
int trigPin  = 9;

float duration;

void setup() {
  Serial.begin(9600);
  leftServo.attach(13);
  rightServo.attach(12);

  forward(10);

  for (int i = 0; i <= 3; i++) {
    turn("right");
    forward(3);
  }
}

void loop() {
  forward(.5);
  float distance = getDistance();
  Serial.println(distance);
  if (distance <= 6) {
    leftServo.detach();
    rightServo.detach();
  }
}

void turn(String direction) {
  if (direction == "right") {
    leftServo.write(1700);
    rightServo.write(1700);
    delay(625);
  } else if (direction == "left") {
    leftServo.write(1300);
    rightServo.write(1300);
    delay(625);
  }
}

void forward(int time) {
  leftServo.write(1700);
  rightServo.write(1300);
  delay(time * 1000);
}

float getDistance() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  duration = pulseIn(echoPin, HIGH);
  float inches = duration * 0.0133 / 2;
  return inches;
}
Workout Tracker Code 
FILENAME = "workouts.txt"

def load_workouts():
    workouts = []
    try:
        with open(FILENAME, "r") as file:
            for line in file:
                line = line.strip()
                if line:
                    parts = line.strip().split(",")
                    if len(parts) != 5:
                        continue
                    workout = {
                        "exercise": parts[0],
                        "sets":     int(parts[1]),
                        "reps":     int(parts[2]),
                        "weight":   int(parts[3]),
                        "date":     parts[4],
                    }
                    workouts.append(workout)
    except FileNotFoundError:
        with open(FILENAME, "w") as file:
            pass
    return workouts

def save_workouts(excercise, sets, reps, weight, date):
    with open(FILENAME, "a") as file:
        file.write(f"{excercise},{sets},{reps},{weight},{date}\n")

def view_workouts(workouts):
    if not workouts:
        print("No workouts found.")
        return
    for workout in workouts:
        print(f"{workout['date']} - {workout['exercise']}: "
              f"{workout['sets']} sets x {workout['reps']} reps @ {workout['weight']} lbs")
    print()

def calculate_total_volume(workouts):
    total_volume = 0
    for workout in workouts:
        total_volume += workout["sets"] * workout["reps"] * workout["weight"]
    print(f"Total Training Volume: {total_volume} lbs")

def add_workout():
    exercise = input("exercise name: ").strip()
    if not exercise:
        print("exercise name cannot be empty.")
        return
    try:
        sets   = int(input("sets: "))
        reps   = int(input("reps: "))
        weight = int(input("weight: "))
    except ValueError:
        print("invalid number inputed try again.")
        return
    date = input("date (YYYY-MM-DD)").strip()
    if not date:
        print("Date cannot be empty.")
        return
    save_workouts(exercise, sets, reps, weight, date)
    print("Workouts saved")

def main():
    while True:
        workouts = load_workouts()
        print("workout tracker")
        print("1. View workouts")
        print("2. Add workout")
        print("3. View total volume")
        print("4. exit")
        choice = input("enter your choice: ").strip()
        if choice == "1":
            view_workouts(workouts)
        elif choice == "2":
            add_workout()
        elif choice == "3":
            calculate_total_volume(workouts)
        elif choice == "4":
            print("thanks for using workout tracker. Stay Strong Stay Hard")
            break
        else:
            print("invalid choice entered. Try again 1-4")

if __name__ == "__main__":
    main()

# Ayaan Patel, 9/15/2025
# Turtles project with more gray shades
from turtle import Turtle, Screen
from PIL import Image

try:
    pixel_size = int(input("Pixel Size (4 to 8): "))
except ValueError:
    print("Invalid Pixel Size default 5 will be used")
    pixel_size = 5

# Luminance converts to greyscale, resize for performance
img = Image.open("car.JPG").convert("L")
img = img.resize((150, 150))

t = Turtle()
t.penup()
t.speed(25)
t.hideturtle()
Screen().tracer(0)
Screen().update()

width  = img.width  * pixel_size
height = img.height * pixel_size / 2

x_offset = -width  // 2
y_offset  =  height // 2

def brightness_to_color(brightness):
    if brightness < 30:  return "black"
    elif brightness < 60:  return "dimgray"
    elif brightness < 90:  return "gray"
    elif brightness < 120: return "darkgray"
    elif brightness < 150: return "silver"
    elif brightness < 180: return "lightgray"
    elif brightness < 210: return "gainsboro"
    else: return None  # skip white background

def draw_image():
    t.penup()
    t.goto(x_offset, y_offset)
    for y in range(img.height):
        print(f"Processing row {y + 1}/{img.height}")
        for x in range(img.width):
            brightness = img.getpixel((x, y))
            color = brightness_to_color(brightness)
            if color:
                t.goto(x * pixel_size + x_offset, -(y * pixel_size) + y_offset)
                t.dot(pixel_size, color)

screen = Screen()
screen.listen()
screen.onkey(draw_image, "s")
screen.onkey(screen.bye, "q")

print("press s to start drawing")
print("press q to quit")

screen.mainloop()
